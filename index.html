📖 All Tracker: Guide for Legends (and Noobs)

🚀 Welcome to the Arena!

Bhai/Behen, agar tu yahan hai, toh matlab tere paas bohot bade goals hain par shayad thodi discipline ki kami hai. Don't worry, Yeh Neon Arena tujhe discipline sikhayega aur agar nahi sikha toh roast toh zaroor karega! 😆

🛠️ 1. Getting Started (Pehle Yeh Kar!)

  • Settings (Ctrl+,): Sabse pehle jaake apna Start Date aur End Date fix kar. Default dates pe mat chor, warna tracker tujhe hamesha 'Loading' hi dikhayega.
  • Column Labels: 'Python', 'DSA' ye toh default hain. Agar tujhe 'Web Dev' ya 'Hacking' pasand hai, toh Settings mein jaake unhe rename karle. It's your world!
  • Beast Mode: Isse on kar agar tujhe thori extra motivation chahiye. Neon glowing vibes milengi! 🌌

📝 2. Daily Tracking (Data Bharna Seekh!)

  • The Tracker Table: Table pe seedha click kar kisi bhi cell pe aur value likh de. Ctrl+S dabane ki zarurat nahi, auto-save hota hai. Smart hai bro!
  • Quick Entry (Ctrl+K): Agar table scroll karne mein alas aa raha hai, toh Ctrl+K daba aur jaldi se data bhar ke nikal le.
  • Bulk Entry: Agar pura week miss kar diya (shame!), toh Bulk Entry use kar ke ek saath saare dino ka record thik karle. Par aage se mat karna! 🤨

🔥 How to Use Neon Arena Like a Legend (Step-by-Step)

01

Niyat Saaf Kar (Setup)

Sabse pehle Settings mein jaake apne dates set kar. Agar end date 2099 hai toh bhai tu amar hai kya? 💀 Realistic dates rakh aur Sync/Login zarur karle warna local storage udd gaya toh tera career bhi udd jayega.

02

Morning Rituals (Routine)

Din shuru hote hi Routine tab pe ja. Wahan tere daily tasks (Exercise, Meditation, etc.) wait kar rahe hain. Unpe checkmark lagana shuru kar. Productivity ka nasha yahin se shuru hoga!

03

Deep Work Mode (Timer)

Jab padhne baitho toh Start Timer button dabao. Category select kar aur phone ko fek de. Jab tak timer chal raha hai, tu duniya ke liye gayab hai. Focus kar, distract mat ho.

04

Log the Grind (Entry)

Padhai khatam? Ab Quick Entry (Ctrl+K) khol. Kitne ghante padha, kitne problems solve kiye, aur kya topics cover kiye—sab likh. Topic mein "Padh liya" mat likhna, details likh taaki baad mein revision mein kaam aaye.

05

Mirror Check (Review)

Hafte ke end mein Heatmap (Ctrl+H) aur Analytics khol ke dekh. Agar chart niche ja raha hai toh mirror mein dekh ke khud ko do-char gaali de aur agle hafte backup kar. 📈

⌨️ 3. Keyboard Shortcuts (Pro Level!)

Ctrl + K Quick Entry (Aaj ka ya purana data)
Ctrl + T Today's Entry (Seedha aaj ka status)
Ctrl + H Heatmap (Dekh kitna green hua hai table)
Ctrl + W Weekly Summary (Pura week kitna padha?)
Ctrl + , Settings (Dates aur Labels badalne ke liye)
Ctrl + F Search (Purana data dhundne ke liye)
Esc Close Modal (Bas band kar de modals!)

📊 4. Analytics & Motivation

Dashboard pe charts aur badges dekhte rehna. Agar "Study Output Trends" niche ja raha hai, toh bhai/behen, phone band kar aur padhne baith. 📉

Badges: Ye achievements tere self-respect ke liye hain. Sab unlock kar aur proud feel kar!

☁️ 5. Cloud Sync (Data Bachao!)

Agar tu chahta hai ki tera data browser crash hone pe bhi safe rahe, toh Sync / Login button pe click kar. Supabase se connect ho ja aur chain ki neend so. 😴

Pro Tip: Roz table ko 'green' dekhne ki koshish kar. Consistency hi tujhe legend banayegi. Ab jaake padh le bsdk! ❤️
